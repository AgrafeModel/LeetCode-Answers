## ğŸš€ Getting Started

This repository is organized by difficulty level to make it easy to find and manage solutions.

### Adding a Solution

1. Navigate to the appropriate difficulty folder (`easy`, `medium`, or `hard`)
2. Create a new file using the naming convention: `{number}-{problem-name}.{ext}`
   - Example: `1-two-sum.py`, `2-add-two-numbers.java`
3. Include the following at the top of your file:
   - Problem number and title
   - Difficulty level
   - Link to the problem
   - Brief description
   - Time and space complexity (optional)

### File Naming Convention

```
{problem-number}-{problem-name}.{extension}
```

Examples:
- `1-two-sum.py`
- `15-3sum.cpp`
- `200-number-of-islands.java`
