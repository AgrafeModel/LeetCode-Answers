# Problems

This folder contains LeetCode problem solutions organized by difficulty level.

## Structure

- `easy/` - Easy difficulty problems
- `medium/` - Medium difficulty problems
- `hard/` - Hard difficulty problems

## Naming Convention

Each solution file should be named using the format:
```
{problem-number}-{problem-name}.{extension}
```

For example:
- `1-two-sum.py`
- `2-add-two-numbers.java`
- `3-longest-substring.cpp`

## Adding Solutions

1. Navigate to the appropriate difficulty folder
2. Create a file with the naming convention above
3. Include the problem description as a comment at the top of the file
4. Add your solution code
5. Optionally, include time and space complexity analysis
